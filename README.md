**SocialSphere** is a modern, full-featured social media application built from the ground up. Functioning as a mini-Twitter clone, it was developed as a key project for the **CodeAlpha Full-Stack Development Internship**. The platform is built using the MERN stack (MongoDB, Express.js, React, Node.js) and features a sleek, responsive user interface styled with Material-UI.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- **Node.js** (v14 or later)
- **npm** (Node Package Manager)
- **MongoDB** (A local instance or a free [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) cloud database)

### Installation & Launch

1.  **Clone the Repository**
    ```sh
    git clone [https://github.com/nikhildhimann/CodeAlpha_SocialMediaPlatform.git](https://github.com/nikhildhimann/CodeAlpha_SocialMediaPlatform.git)
    cd CodeAlpha_SocialMediaPlatform
    ```

2.  **Set Up the Backend (`/server`)**
    ```sh
    # Navigate to the server directory
    cd server

    # Install dependencies
    npm install

    # Create a .env file and add your variables
    # (MONGO_URI, JWT_SECRET, PORT)
    ```
    Then, start the server:
    ```sh
    node index.js
    ```

3.  **Set Up the Frontend (`/client`)**
    *Open a new terminal for this step.*
    ```sh
    # Navigate to the client directory
    cd client

    # Install dependencies
    npm install
    ```
    Then, start the React application:
    ```sh
    npm start
    ```
    The application will be live at `http://localhost:3000`.
